1. 常用命令

   1. 初始化git项目常用命令

      ```bash
      #移动到项目根路径
      cd project-root-path
      
      #初始化git工程
      git init
      
      #提交到暂存区
      git add README.md Text.md
      
      #查看git提交状态
      git status
      
      #提交到本地仓库
      git commit -m "注释"
      
      #重命名本分支名称为main
      git branch -M main
      
      #将该仓库和远程git@github.com:Java-panda/gitexe.git进行绑定
      git remote add origin git@github.com:Java-panda/gitexe.git
      
      
      ```
